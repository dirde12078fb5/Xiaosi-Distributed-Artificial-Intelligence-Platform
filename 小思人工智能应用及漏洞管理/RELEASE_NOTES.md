# 小思人工智能应用及漏洞管理 v1.0.0 发布说明

## 📢 发布概述

小思人工智能应用及漏洞管理模块首次发布！这是小思分布式人工智能平台的核心安全管理组件，集成了AI应用管理与安全漏洞扫描两大核心功能。

---

## ✨ 新功能一览

### 🎯 五大核心功能模块

| 模块 | 功能说明 |
|------|---------|
| **仪表盘** | 全局概览，统计卡片 + 系统信息 + 活动日志 |
| **AI应用管理** | 模型注册、状态监控、版本管理、配置持久化 |
| **系统漏洞扫描** | 快速/完整/深度三种模式，18+安全检测项 |
| **网络端口扫描** | 自定义端口范围，服务识别，三档速度 |
| **AI模型安全** | 对抗性检测、数据泄露、完整性、隐私合规 |

### 🔧 技术亮点

- **多线程架构**：扫描任务后台运行，UI流畅不卡顿
- **现代化UI**：卡片式设计，蓝色主题，与平台风格统一
- **配置持久化**：JSON格式保存，数据不丢失
- **报告导出**：支持TXT和JSON双格式
- **跨平台支持**：Windows / Linux / macOS

## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动程序

```bash
python 小思人工智能应用及漏洞管理.py
```

### 系统要求

- **Python**: 3.8+（推荐 3.14.1）
- **内存**: 2GB+
- **磁盘**: 100MB+ 可用空间
- **系统**: Windows 10/11, Linux, macOS

## 📊 功能详情

### 1. 仪表盘

- 4个统计卡片：AI模型数量、运行中应用、已发现漏洞、安全评分
- 系统信息展示：OS、CPU、内存、主机名
- 实时活动日志

### 2. AI应用管理

- 支持模型类型：LLM、CV、语音、多模态、其他
- 模型增删改查
- 状态自动检测
- JSON配置持久化

### 3. 系统漏洞扫描

| 扫描模式 | 检测项数量 | 适用场景 |
|---------|-----------|---------|
| 快速扫描 | 5项 | 日常快速检查 |
| 完整扫描 | 10项 | 常规安全评估 |
| 深度扫描 | 18项 | 全面安全审计 |

**检测类别**：系统、安全、服务、账户、网络、应用、设备、硬件

**风险等级**：严重(critical)、高危(high)、中危(medium)、低危(low)

### 4. 网络端口扫描

- 端口范围：1 - 65535
- 扫描速度：慢速 / 正常 / 快速
- 内置30+常见服务识别
- 实时进度反馈

### 5. AI模型安全检测

- 对抗性攻击检测
- 数据泄露风险评估
- 模型完整性校验
- 隐私合规检测
- 安全评分（0-100分）
- 修复建议输出
