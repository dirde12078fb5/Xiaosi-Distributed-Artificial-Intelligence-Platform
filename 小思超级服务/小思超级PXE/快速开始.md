# 🚀 小思超级PXE - 快速开始指南

## 一键启动

**最简单的方式：**
```bash
python quick_start.py
```

选择选项 3，自动运行诊断并启动GUI！

## 常见问题解答 (FAQ)

### ❓ 问题 1: DHCP 客户端收不到 IP

**可能原因：**
- 没有管理员/root权限
- 端口被占用
- 防火墙阻止
- 网络中有其他DHCP服务器

**解决方法：**
1. 运行诊断工具：`python diagnose_advanced.py`
2. 检查是否以管理员/root身份运行
3. 确保没有其他DHCP服务在运行

### ❓ 问题 2: 无法绑定端口 67/69

**Windows:**
- 以管理员身份运行命令提示符
- 或右键点击选择"以管理员身份运行"

**Linux:**
```bash
sudo python gui.py
```

**macOS:**
```bash
sudo python3 gui.py
```

### ❓ 问题 3: 端口被占用

**Windows:**
```powershell
netstat -ano | findstr :67
```
找到PID后，在任务管理器中结束进程

**Linux:**
```bash
sudo netstat -ulpn | grep :67
```
或者检查dnsmasq或dhcpcd服务

### ❓ 问题 4: 配置文件在哪？

配置文件是：`config.json`

如果不存在，程序会自动从 `config.example.json` 复制一份

## 使用步骤

### 步骤 1: 运行诊断

首先运行诊断，检查环境：
```bash
python diagnose_advanced.py
```

### 步骤 2: 配置网络

编辑 `config.json` 中的 `server_ip`，设置为你本机的IP

### 步骤 3: 启动GUI

```bash
python gui.py
```

### 步骤 4: 启动服务器

在GUI中点击"▶️ 启动服务器"
